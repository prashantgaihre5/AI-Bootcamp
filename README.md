# AI-Bootcamp

This repository contains my first set of assignments for the AI Boot Camp. These programs were written to demonstrate foundational programming skills in Python, focusing on core logic and fundamental algorithms.

## Assignments Included
1. **Prime Checker (`prime_checker.py`)** - Determines whether a given input number is a prime number.
2. **Palindrome Checker (`palindrome.py`)** - Checks if a string or number reads the same forwards and backwards.
3. **Even or Odd (`even_odd.py`)** - A simple program that checks if an integer is even or odd.
4. **Matrix Multiplication (`matrix_product.py`)** - Calculates the product of two matrices.
5. **Factorial Calculator (`factorial.py`)** - Computes the factorial of a non-negative integer.

## How to Run
Ensure you have Python installed, then run any of the scripts via the command line:
```bash
python assignments/prime_checker.py
```
